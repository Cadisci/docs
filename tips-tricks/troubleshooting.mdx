When using Lovable, you may occasionally encounter technical issues. Below are common problems and recommended actions to troubleshoot and resolve them.

<AccordionGroup>
  <Accordion title="Persistent 'Fix Error' Loops">
    <b>Problem:</b> Repeated failures where errors persist despite multiple attempts to fix them.
    <b>Solution:</b>

    * Approach with creativity: Adjust your input and prompts to help the AI understand the changes you’re trying to make.

    * Use Chat Mode: Engage the AI in chat mode for step-by-step guidance and tailored suggestions to break the loop.
  </Accordion>

  <Accordion title="Setting Environment Variables">
    <b>Problem:</b> Issues may arise when configuring environment variables, particularly with Supabase Edge Functions and Secrets.
    <b>Solution:</b>

    * Double-check the environment variable syntax and names.

    * Refer to the [Supabase Docs](https://supabase.com/docs) for guidance.
  </Accordion>

  <Accordion title="HTTP Fetch Failures">
    <b>Problem:</b> Errors during HTTP fetch requests can disrupt authentication or degrade app performance.
    <b>Solution:</b> Copy and paste the exact error message into the chat to get context-specific advice from the AI.
  </Accordion>

  <Accordion title="Freezing or Loading Failures">
    <b>Problem:</b> Editor pages freeze, fail to load, or do not reflect the latest updates.
    <b>Solution:</b> Report this as a bug via our [feedback portal](https://lovable.dev/support).
  </Accordion>

  <Accordion title="Rollback Feature Fails">
    <b>Problem:</b> Unable to revert to a previous state of the project.
    <b>Solution:</b> Submit a bug report via our [feedback portal](https://lovable.dev/support).
  </Accordion>

  <Accordion title="Cloud Deployment Issues">
    <b>Problem:</b> Apps fail to redeploy or get stuck during deployment.
    <b>Solution:</b> File a bug report via our [feedback portal](https://lovable.dev/support).
  </Accordion>

  <Accordion title="AI-Related Errors">
    <b>Problem:</b> The AI is hallucinating or not resolving issues correctly.
    <b>Solution:</b>

    * Revert to a stable version of your application.

    * Improve future prompts:

      * Be specific and clear. Refer to our [Prompting Handbook]().

      * Use the select tool for precise edits.

      * Provide context by adding relevant documentation.
  </Accordion>

  <Accordion title="Refactor Broke Everything in My Code">
    <b>Problem:</b> Code refactor caused major issues.
    <b>Solution:</b>

    * Revert to a stable version of your application.

    * Make smaller, incremental changes.

    * Check your codebase history and debug using tools like ChatGPT.
  </Accordion>

  <Accordion title="AI Is Not Responding">
    <b>Problem:</b> The AI chat is stuck.
    <b>Solution:</b> Disconnect and reconnect Supabase or restore an earlier stable version.
  </Accordion>

  <Accordion title="Preview Shows a Blank Screen">
    <b>Problem:</b> The preview displays a white or blank screen.
    <b>Solution:</b> Revert to a stable version and check the console logs for relevant errors.
  </Accordion>

  <Accordion title="Taking Your Projects Offline">
    <b>Problem:</b> Unable to unpublish or deactivate projects.
    <b>Solution:</b> Upvote this feature on our [Feature Request Board](https://lovable.featurebase.app/).
  </Accordion>

  <Accordion title="Being Kicked Out of a Project">
    <b>Problem:</b> Unexpected logout or access loss.
    <b>Solution:</b> Contact our support engineers directly.
  </Accordion>

  <Accordion title="Loading Errors">
    <b>Problem:</b> Loading issues or downtime.
    <b>Solution:</b> Contact our support engineers.
  </Accordion>

  <Accordion title="Error: Stream Interrupted">
    <b>Problem:</b> Stream interrupted due to invalid Supabase credentials.
    <b>Solution:</b> Check and update your Supabase credentials in the settings.
  </Accordion>

  <Accordion title="Edge Function Errors">
    <b>Problem:</b> Edge function errors.
    <b>Solution:</b> Supabase logs integration is coming soon. Meanwhile, check the logs and paste errors into Lovable's AI code editor.
  </Accordion>

  <Accordion title="Stripe Errors">
    <b>Problem:</b> Issues with payment processing.
    <b>Solution:</b> Try a different bank or card. If the issue persists, contact support.
  </Accordion>

  <Accordion title="GitHub Issues">
    <b>Problem:</b> Connection issues after changing your GitHub username or repository name.
    <b>Solution:</b> Revert to the old names. If the repo was deleted, restore it using [this guide](https://docs.github.com/en/repositories/creating-and-managing-repositories/restoring-a-deleted-repository).
  </Accordion>

  <Accordion title="Sandbox Mode for Testing">
    <b>Problem:</b> Users want a safe, non-production environment.
    <b>Solution:</b> This feature is not currently available. Upvote it on our [Feature Request Board](https://lovable.featurebase.app/).
  </Accordion>

  <Accordion title="Supabase Connection Lost">
    <b>Problem:</b> Lost connection to Supabase.
    <b>Solution:</b> Disconnect and reconnect Supabase.
  </Accordion>

  <Accordion title="Project Not Found">
    <b>Problem:</b> Unable to find your project.
    <b>Solution:</b> Disconnect and reconnect Supabase.
  </Accordion>

  <Accordion title="Unable to Switch to Main Branch">
    <b>Problem:</b> Errors when switching to the main branch.
    <b>Solution:</b> Disconnect and reconnect Supabase.
  </Accordion>

  <Accordion title="Remix Feature Does Not Work">
    <b>Problem:</b> Remixing your project does not work.
    <b>Solution:</b> Note that projects connected to Supabase **cannot** be remixed for security reasons. If this is not the case, contact support.
  </Accordion>

  <Accordion title="Challenges with Large Projects">
    <b>Problem:</b> Large projects encounter performance issues.
    <b>Solution:</b> We’re optimizing AI performance for large projects. Stay tuned for updates.
  </Accordion>

  <Accordion title="Need More Help?">
    If you encounter any other issues, reach out via our [feedback portal](https://lovable.dev/support) or explore our [prompting guide](/tips-tricks/prompting#debugging-and-issue-reporting).
  </Accordion>
</AccordionGroup>
