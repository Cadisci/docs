## Product Capabilities

**Lovable** currently excels at building front-end applications. However, using our Supabase integration  will allow you to build full-stack web apps mostly within Lovable's UI.

| Capability     | Description                      | Maturity Level            |
| -------------- | -------------------------------- | ------------------------- |
| Frontend / UI  | Build user interfaces & frontend | 游릭 Mature                 |
| Persistence    | Store and retrieve data          | 游리 Requires some patience |
| Authentication | Handle user login and accounts   | 游리 Requires some patience |
| AI functions   | Integrate LLM calls              | 游리 Requires some patience |
| Checkout       | Receive payments via Stripe      | 游리 Requires some patience |
