## Verified Integrations

Verified integrations work best with Lovable:&#x20;

* [**Builderio**](https://lovable.dev/blog/2025-01-22-figma-to-lovable-builder-io-native-integration): Turn Figma designs into full stack applications within Lovable.

* [Runware](https://lovable.dev/integrations/runware): Make your app generate images from text prompts.

* [OpenAI](https://lovable.dev/integrations/openai): Add AI capabilities to your app using OpenAI's powerful language models.

* [Anthropic](https://lovable.dev/integrations/anthropic): Add powerful AI capabilities to your app using Anthropic's Claude models.

* [Stripe](https://lovable.dev/integrations/stripe): Add payment processing capabilities to your app.

* [Three.js](https://lovable.dev/integrations/threejs): Add interactive 3D graphics to your app.

* [Resend](https://lovable.dev/integrations/resend): Add email capabilities to your app using Resend's modern email API.

* [D3.js](https://lovable.dev/integrations/d3js): Create dynamic, interactive data visualizations.

* [Highcharts](https://lovable.dev/integrations/highcharts): Create professional, interactive charts and visualizations. &#x20;

* [p5.js](https://lovable.dev/integrations/p5js): Create creative coding projects and interactive graphics. [Examples](https://x.com/lovable_dev/status/1869765295931527488) here.

* [**Replicate**](https://replicate.com/): Run and fine-tune open-source AI models.

* [**Stability AI:**](https://stability.ai/) **D**evelops open AI models across various modalities, including image, language, audio, video, 3D, and biology.

* [**Rive**](https://rive.app/)**: R**eal-time interactive design tool that allows designers and developers to create and ship interactive animations

* [**Together.ai**](https://together.ai/)**: H**elp families and teams improve mental health and well-being through connected conversations.

* [**PostHog**](https://posthog.com/): Open-source product analytics platform that provides tools for analyzing user behavior, including session replays, feature flags, and A/B testing.

* [Mapbox](https://t.co/QThcnZclS9): Powers navigation for people, packages, and vehicles everywhere.

* [Firecrawl](https://t.co/31mgUJJ3VO): Turn websites into LLM-ready data.

* [Perplexity](https://t.co/hpxxMepIkW): Add powerful AI capabilities to your app.

* [Zapier](https://t.co/7kt8E7ApgC): Automation tool that connects different web applications.

* [**GoJS**](https://x.com/lovable_dev/status/1869700306583613486): Add flowcharts especially useful for visualizing workflows and processes.

* [Transformers.js (Hugging Face):](https://huggingface.co/docs/transformers.js/) Run state-of-the-art machine learning models directly in the browser without the need for a server.

* [xAI:](https://x.ai/) AI scheduling tool that helps users automate meeting scheduling.

* [Twilio:](https://www.twilio.com/) Make and receive phone calls, send and receive text messages, and perform other communication functions.

* [Google Workspace](https://workspace.google.com/): Cloud-based productivity and collaboration tools developed by Google.

* [Lemon Squeezy:](https://www.lemonsqueezy.com/) E-commerce platform that simplifies selling digital products.

* [Serper.dev](https://serper.dev/): Integrate Google Search results into your applications.

* [Notion](https://www.notion.so/): Note-taking, task management, and database functions.

* [Mailgun](https://www.mailgun.com/): Send, receive, and track emails.

* [Calendly:](https://calendly.com/) Scheduling tool that simplifies meeting planning.

* [SendGrid:](https://sendgrid.com/) Email delivery service for transactional and marketing emails management & delivery.
